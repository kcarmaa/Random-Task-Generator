# Генератор случайных задач
Создатель: Михаил Рыбин

Программа имеет следующие функции: 
1. Добавление новой задачи
2. Получение случайной задачи из списка
3. Просмотр истории случайных задач в виде списка
4. Имеется возможность фильтрации задач по категориям(дела по дому, учеба, спорт и работа)
5. Также программа автоматически сохраняет списки задач и историю случайных задач в json-файлы
